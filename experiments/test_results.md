# Experimental Validation

Bench testing was performed using high-speed camera analysis to evaluate response time and control accuracy.

## Test Method

The thrust vector control mechanism was actuated using an Arduino-driven servo system. Motion was recorded at 120 FPS and analyzed frame-by-frame to estimate response time and angular accuracy.

## Results

- Average response time: 44.5 ms
- Maximum deflection: ±5°
- Control accuracy: ±0.2°
- Video analysis frame rate: 120 FPS

## Interpretation

The results demonstrate that a low-cost thrust vector control system can achieve measurable and repeatable angular response suitable for prototyping and educational research.

## Notes

These results represent bench-testing performance under controlled conditions and are intended for prototype validation.
