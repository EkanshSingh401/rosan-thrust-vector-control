# ROSAN Control Notes

This folder contains a simplified public-facing MATLAB model for thrust vector control response analysis.

## Purpose

The model demonstrates the type of response analysis used to evaluate:

- target angle tracking
- control response timing
- actuator behavior
- feasibility of low-cost control hardware

## Modeling Approach

A simplified second-order response model is used to approximate closed-loop actuator behavior for thrust vector deflection.

## Notes

- This is a lightweight public demo model.
- It is intended to illustrate the simulation workflow described in the project.
- It is not a full release of the original Simulink development environment.
